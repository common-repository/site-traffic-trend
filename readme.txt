== Description ==

Adds rankarchive.com trend to the admin Dashboard for current domain. The trend shows
Alexa traffic rank history for period makes it convenient to see how is your site grows.

**Requirements:**

*   WordPress 2.9++
*   PHP 5.1.xx
*   MySql 5.1.xx
*   PHP 4.xx and MySql 4.xx not supported
*   Free one-click domain submission to http://rankarchive.com

= Instruction =

1. Upload the full directory into your wp-content/plugins folder
2. Activate the plugin at the plugin administration page
3. Go to http://www.rankarchive.com site and enter the domain name of your blog
4. If your domain is new, the statistic start showing after two days
5. If your doamin already included you will see the graph in the Dashboard widget

= Features =

* Automatically adds the Traffic Rank widget to the admin Dashboard.
* Shows traffic changes collected by rankarchive for your domain.
* Statistic graph grows with the site traffic, not like Alexa shows it.
* Stores traffic data at the external source, not in your database.

== Installation ==

== Frequently Asked Questions ==

= Will it slow down my site ? =

No, the plugin works once when admin visits Dashboard, it is not using your the database also.
